# FileStructure that is recommended
- src/
    - index.js / server.js / main.js [Entry Point , can be of any name]
    - app [Congig,UrlEncode,Cookies,etc]
    - constant [enums,DB-name,etc]
    - DB/
    - Models/
    - Controller/
    - Routes/
    - Middlewares
    - Utils/
    - More/ (Depends)
